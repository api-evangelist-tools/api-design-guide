API Stylebook (Builder)

I was pretty stoked when my friend Arnaud Lauret developed the API Stylebook. I took the YAML core of the design topics he's aggregated across the API design guides he's indexed, and started working with an editor that runs on Github, and will allow anyone to fork, and use to build their own API Stylebook using his work.

The JavaScript on this page loads the _data/topics.yaml from this projects Github repository. If you fork this project, and enter a valid Github personal token, you should be able to save the results.

This is just a prototype. You are welcome to fork, but I will keep working on it, adding autocomplete suggestions based upon other API design guides, and the ability to ignore some elements. Maybe I'll also work on allowing the addition of elements to the design guide--both categories and elements.

I will add these to the Github issues for the project. If you have any feedback let me know there.
