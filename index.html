API Stylebook (Builder)

I was pretty stoked when my friend Arnaud Lauret developed the API Stylebook. I took the YAML core of the design topics he's aggregated, and started developing this editor that runs 100% on Github, so that anyone to fork, and use to build their own API Stylebook using Arnaud's work.

Currently there are two screens for this tool:

  • Editor - Allows for the editing of the YAML API Design Stylebook stored in this Github repository.
  • View - Allow syou to view the API Design Stylbook stored in this Github repository.

This is just a prototype. You are welcome to fork, but I will keep working on it, adding autocomplete suggestions based upon other API design guides, and the ability to ignore some elements. Maybe I'll also work on allowing the addition of elements to the design guide--both categories and elements.

I will add these to the Github issues for the project. If you have any feedback, or suggestions let me know.
